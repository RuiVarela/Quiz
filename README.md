# Quiz
Quiz is a cross platform "quiz" presenter powered by [OpenDB](https://opentdb.com/).  
This is also a small enough app, that can be used as Kotlin Multiplatform starting point

Supported Platforms
- Windows (JVM)
- Mac (JVM)
- iOS
- Android
